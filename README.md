# Squiggly and Frens Blind Box Program

## Project description
A simple beginner program for buying and selling blind boxes from the fictional brand, "Squiggly and Frens"
* This project does not have a Save / Load System

## Features
- Wallet: Check current total and type in promo codes (Use FREEMONEYGLITCH for a free $100)
- Shop: Buy from the two Squiggle and Frens blind boxes available
- Collection: Manage your Unopened Blind boxes by opening or selling them. Manage your Opened Boxes by selling them or checking their rarity

## How to Run
1. Ensure that you have both Java JDK and JavaFX installed on your computer
2. Save my program folder to your computer (ensure you've downloaded all the files)
3. Open a terminal (Preferably in VS Code)
4. Navigate to whatever folder contains the file
5. Compile the program by typing: javac TitlePage.java
6. Run my program by typing: java TitlePage
7. Click start and navigate between the three screens available to you through the main menu
